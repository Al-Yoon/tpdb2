# MercadoLibre

## Descripción

Este proyecto simula una plataforma de comercio electrónico similar a MercadoLibre, El sistema está construido con un enfoque de bases de datos híbridas para optimizar la consulta, almacenamiento y análisis de datos de acuerdo a sus características y necesidades específicas.

### Tecnologías Utilizadas
- **PostgreSQL**: Para la gestión de datos estructurados y transaccionales.
- **MongoDB**: Para almacenar datos no estructurados o semi-estructurados, como perfiles de usuario, comentarios, productos con descripciones flexibles, etc.
- **Neo4j**: Para gestionar relaciones complejas entre usuarios, productos, categorías, y realizar recomendaciones basadas en grafos.

### Requisitos Previos

- PostgreSQL 13 o superior
- MongoDB 4.x o superior
- Neo4j 4.x o superior
